Dynamic Programming — Bài 1
Fibonacci DP
Tính số Fibonacci thứ n bằng Tabulation (bottom-up). Minh họa nguyên lý cốt lõi của DP: lưu kết quả subproblem để không tính lại.
Tại sao cần DP? — Đệ quy thuần F(5) sẽ tính F(3) hai lần, F(2) ba lần... → O(2ⁿ). DP lưu lại → O(n).
1 Đặt dp[0] = 0, dp[1] = 1 (base case)
2 Với mỗi i từ 2 → n: dp[i] = dp[i-1] + dp[i-2]
3 Kết quả là dp[n]. Mỗi ô chỉ tính một lần.
Bảng dp[] — F(0) → F(n)  ■ target  ■ dp[i-2]  ■ dp[i-1]
Code — Tabulation (bottom-up)
def fibonacci(n): if n <= 1: return n dp = [0, 1] # base cases for i in range(2, n + 1): dp.append(dp[i-1] + dp[i-2]) # ← công thức DP return dp[n] # Space có thể tối ưu → O(1) bằng 2 biến: # a, b = 0, 1 # for _ in range(n): a, b = b, a + b

Dynamic Programming — Bài 2
Knapsack 0/1 — Combo Khuyến Mãi
Chọn combo sản phẩm tối ưu trong giới hạn ngân sách. Mỗi item chỉ dùng một lần (0 = không chọn, 1 = chọn).
Logic vận hành: Xây bảng 2D — hàng = item, cột = ngân sách từ 0 → W.
1 Với mỗi item i, duyệt mọi ngân sách w từ 0 → W.
2 Không chọn item i: dp[i][w] = dp[i-1][w]
3 Chọn item i (nếu w ≥ weight[i]): dp[i][w] = dp[i-1][w - weight[i]] + value[i]
4 Lấy max của 2 lựa chọn. Traceback để tìm item được chọn.
Chọn / bỏ sản phẩm (click để toggle)
Bảng DP 2D — dp[item][budget]  ■ traceback path  ■ giá trị > 0
Code — Knapsack 0/1
def knapsack(weights, values, W): n = len(weights) dp = [[0] * (W + 1) for _ in range(n + 1)] for i in range(1, n + 1): for w in range(W + 1): # Không chọn item i dp[i][w] = dp[i-1][w] # Chọn item i (nếu đủ túi) if w >= weights[i-1]: dp[i][w] = max( dp[i][w], dp[i-1][w - weights[i-1]] + values[i-1] ) return dp[n][W] # max value

Hashing — Bài 3
Subarray Sum = K
Đếm số dải con có tổng bằng k. Dùng prefix sum + hash map để giải O(n) thay vì O(n²) nested loop.
Insight chính: Nếu prefixSum[j] - prefixSum[i] = k, thì arr[i+1..j] là dải cần tìm.
1 Duyệt từng phần tử, cộng vào prefixSum đang chạy.
2 Với mỗi bước: kiểm tra xem prefixSum - k đã xuất hiện trong map chưa?
3 Nếu có → tìm thấy một dải! Cộng số lần xuất hiện vào count.
4 Lưu prefixSum hiện tại vào map (tần suất). map[0]=1 để xử lý dải bắt đầu từ index 0.

Code — Prefix Sum + HashMap
def subarray_sum(nums, k): map_ = {0: 1} # prefixSum → tần suất prefix_sum = 0 count = 0 for num in nums: prefix_sum += num # Nếu (prefix_sum - k) đã có trong map # → có dải con có tổng = k count += map_.get(prefix_sum - k, 0) # Ghi nhớ prefix_sum này map_[prefix_sum] = map_.get(prefix_sum, 0) + 1 return count
